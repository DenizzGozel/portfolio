Deniz Gozel - Portfolio Website

Overview:
This repository contains the source code for my personal portfolio website, showcasing my projects, skills, and contact information 
in a clean, responsive, one-page layout.


Tech Stack:
* HTML5
* CSS3 (Bootstrap, Animate.css, Flexslider)
* JavaScript (jQuery, Typed.js, Isotope.js)
* FontAwesome icons
* Google Fonts (Nunito Sans)
* Formspree for contact form handling


Features:
* Responsive single-page design with smooth scrolling and scrollspy navigation
* Sections: Home, Portfolio, About Me, Services, Contact
* Filterable portfolio gallery powered by Isotope.js
* Animated typing effect for skill highlights via Typed.js
* Downloadable resume (assets/resume\_Deniz\_Gozel.pdf)
* Contact form integrated with Formspree
* Social media links and dynamic copyright year


Prerequisites:
* A modern web browser (Chrome, Firefox, Safari, Edge)


Project Structure:
```
/ (root)
├─ index.html            # Main entry point, contains all sections
├─ css/                  # Stylesheets
│  ├─ bootstrap.css      # Bootstrap framework
│  ├─ animate.css        # CSS animations
│  ├─ flexslider.css     # Slider styles
│  ├─ style.css          # Core template styles
│  └─ custom.css         # Custom overrides and tweaks
├─ js/                   # JavaScript files
│  ├─ vendor/            # Third‑party libraries (jQuery, Bootstrap, etc.)
│  └─ custom.js          # Site-specific scripts (navigation, animations)
├─ fonts/                # Icomoon icon font files
├─ images/               # All site images and project screenshots
├─ assets/               # Downloadable assets (e.g., resume)
│  └─ resume_Deniz_Gozel.pdf
└─ README.txt            # This file
```

Usage:
* Use the navbar or scroll to navigate between sections.
* Click portfolio filters (All, Python, React, Node.js, JS/HTML/CSS) to refine projects.
* Click project thumbnails to visit live demos in a new tab.
* Download the CV by clicking the "Download CV" button.
* Send a message via the contact form; submissions go through Formspree.
